# 🩺 Premium Radiology Portfolio & CMS

Welcome to your state-of-the-art medical portfolio. This platform is designed to showcase your clinical expertise with a premium, high-tech aesthetic and real-time data management.

## 🚀 Quick Start
To view your website, simply open `index.html` in any modern web browser or use a local live server.

## ⚙️ Content Management (Admin Panel)
You can update your entire portfolio without touching a single line of code!
1.  Navigate to your website URL.
2.  Add `/#admin` to the end of the URL (e.g., `localhost:8000/#admin`).
3.  Login with your secure password.
4.  **Manage Everything**: Edit your profile, add new clinical cases, update your experience, and manage patient reviews.

## ☁️ Real-time Cloud Sync
This portfolio is integrated with **Firebase Realtime Database**.
- **Global Updates**: Any change you make in the Admin Panel is instantly saved to the cloud and visible to everyone visiting your site.
- **Sync Status**: Look for the "Syncing with Cloud... ☁️" notification in the Admin Panel to confirm your changes are live globally.

## 📂 Project Structure
- `index.html`: The main entry point.
- `js/app.js`: The "brain" of the application (Logic, Store, and Cloud Sync).
- `css/style.css`: The "beauty" (Premium Glassmorphism 2.0 theme).
- `image/`: Your medical images and hero visuals.

## 🔄 Updating to GitHub
If you want to update the source code files on GitHub:
1.  **Manual Upload**: Drag and drop the `js/app.js`, `css/style.css`, and any new images from the `image/` folder into your GitHub repository in your browser.
2.  **Auto-Sync**: If you have Git installed, you can double-click `sync.bat` to automatically push all local changes to your repository.
